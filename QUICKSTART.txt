У ТЕБЯ УЖЕ УСТАНОВЛЕНО:
✅ Python 3.11
✅ PostgreSQL 16 (БД hh_fast_bot создана)
✅ Redis (работает)
✅ Виртуальное окружение (venv)

ОСТАЛОСЬ:
=========

1️⃣ РАСПАКУЙ ВСЕ ФАЙЛЫ (ты их скачал)

Скачал файлы из Claude.ai? Распакуй их в папку проекта:

Windows:
  C:\Users\YourName\hh_fast_bot\
  
Linux/macOS:
  ~/hh_fast_bot/


2️⃣ СОЗДАЙ СТРУКТУРУ ПАПОК

Открой cmd/PowerShell в папке hh_fast_bot и выполни:

Windows:
```
mkdir handlers
mkdir keyboards  
mkdir services
mkdir logs
```

Linux/macOS:
```
mkdir -p handlers keyboards services logs
```


3️⃣ РАССТАВЬ ФАЙЛЫ ПО МЕСТАМ

Вот какие файлы где должны быть:

КОРЕНЬ (hh_fast_bot/):
- main.py
- config.py
- models.py
- partners_config.py
- .env (уже есть!)
- requirements.txt
- README.md

ПАПКА handlers/:
- __init__.py (пусто)

- search.py (из handlers_search.py)
- filters.py (из handlers_filters.py)

ПАПКА keyboards/:
- __init__.py (пусто)
- search_keyboards.py

ПАПКА services/:
- __init__.py (пусто)
- hh_service.py
- db_service.py
- cache_service.py
- formatter.py

ПАПКА logs/:
- (пусто, сюда будут логи)


4️⃣ ПРОВЕРЬ ВСЕ УСТАНОВЛЕНО

Открой cmd в папке hh_fast_bot:

```bash
# Активируй виртуальное окружение
venv\Scripts\activate  # Windows
# ИЛИ
source venv/bin/activate  # Linux/macOS

# Установи зависимости
pip install -r requirements.txt

# Должно вывести:
# Successfully installed aiogram-3.3.0 asyncpg-0.29.0 redis-5.0.0 ...
```

6️⃣ ТЕСТИРУЙ В TELEGRAM

1. Открой Telegram
2. Найди бота (@HH_Fast_Bot или как его называешь)
3. Нажми /start
4. Выбери "🔍 Начать поиск"
5. Выбери город → опыт → график → и т.д.
6. Должны появиться вакансии!

ПЕРВАЯ ВАКАНСИЯ НА КАЖДОЙ СТРАНИЦЕ = ТВОЯ РЕКЛАМА 💰


7️⃣ ОТРЕДАКТИРУЙ ПАРТНЁРСКИЕ ВАКАНСИИ

Открой файл partners_config.py и замени демо-вакансии на свои:

```python

    "Москва": [
        {
            "emoji": "🔥",
            "title": "Твоя вакансия",
            "salary": "200 000 ₽",
            "schedule": "Полная занятость • Москва",
            "description": "Описание",
            "button_text": "Устроиться →",
            "url": "https://твоя-ссылка.com"  # ← ЗАМЕНИ НА СВОЮ!
        }
    ]
}
```

Сохрани файл, бот обновит данные при следующем запросе.


8️⃣ ОСТАВЬ БОТА РАБОТАТЬ

```bash
python main.py
```

Оставь cmd окно открытым - бот будет работать 24/7 пока оно открыто!

❓ ЕСЛИ ОШИБКА:

❌ "ModuleNotFoundError: No module named 'handlers'"
✅ Создай папку handlers с файлом __init__.py

❌ "BotToken is empty"
✅ Проверь .env - BOT_TOKEN заполнен?

❌ "Connection refused"
✅ Включи PostgreSQL и Redis:
   - PostgreSQL: уже запущена
   - Redis: redis-cli ping (должно вывести PONG)

❌ "No such file or directory: 'logs/bot.log'"
✅ Создай папку: mkdir logs

❌ Бот не отвечает в Telegram
✅ Проверь:
   - BOT_TOKEN правильный в .env?
   - cmd окно с python main.py открыто?
   - Логи не показывают ошибок?
