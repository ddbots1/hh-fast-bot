✏️ ФАЙЛЫ, КОТОРЫЕ НУЖНО ОТРЕДАКТИРОВАТЬ
========================================

После создания структуры проекта отредактируй эти файлы:


1️⃣ .env (КРИТИЧНО!)
====================

Открой файл .env и ЗАМЕНИ:

❌ БЫЛО:
BOT_TOKEN=8229276656:AAFeBfH9eVKNKge1JG-SvNegmp40IuCq1pA

✅ ЗАМЕНИ НА:
BOT_TOKEN=твой_новый_токен_от_botfather

❌ БЫЛО:
DB_PASSWORD=0319222M@x

✅ ЗАМЕНИ НА:
DB_PASSWORD=твой_пароль_postgres

Остальные значения можно оставить как есть (если ничего не менял при установке).


2️⃣ partners_config.py (ГЛАВНОЕ ДЛЯ МОНЕТИЗАЦИИ!)
===================================================

Это файл с партнёрскими вакансиями (реклама).

ТЕКУЩИЕ ДЕМО-ВАКАНСИИ:
- Курьер на скутере (Москва)
- Менеджер по продажам (Москва)
- Продавец-консультант (СПб)
- Junior Python разработчик (Казань)
- Общая вакансия (для всех городов)

ОТРЕДАКТИРУЙ:
1. Замени текст должностей на твои
2. Замени ссылки в "url" на твои партнёрские ссылки
3. Обнови зарплату и описания
4. Добавь свои города в словарь

ПРИМЕР:
```python
"Москва": [
    {
        "emoji": "💰",
        "title": "Твоя вакансия здесь",
        "salary": "1 000 000 ₽",
        "schedule": "Полная занятость • Москва",
        "description": "Описание твоей партнёрской вакансии",
        "button_text": "Устроиться →",
        "url": "https://твоя-партнёрская-ссылка.ru"
    }
]
```

ВАЖНО: На каждой странице поиска первая вакансия будет одной из этих!


3️⃣ config.py (ОПЦИОНАЛЬНО)
===========================

Тут настройки бота. Можешь изменить:

```python
VACANCIES_PER_MESSAGE = 3          # Количество вакансий за раз
PARTNER_POSITION = 0                # На какой позиции партнёрка (0 = первая)
ANTI_FLOOD_DELAY = 0.5              # Задержка между сообщениями (в сек)
HH_VACANCIES_PER_PAGE = 20          # Максимум вакансий за запрос к HH.ru
```

Остальное менять не советую, если не знаешь что делаешь!


4️⃣ main.py (НЕ МЕНЯТЬ)
======================

Главный файл запуска. Менять не надо, если всё работает.

Если хочешь добавить новые обработчики, добавь импорт:

```python
from handlers import new_handler
dp.include_router(new_handler.router)
```


5️⃣ ДРУГИЕ ФАЙЛЫ (НЕ ТРОГАТЬ)
=============================

✅ models.py - модели данных (не менять)
✅ hh_service.py - API HH.ru (не менять)
✅ db_service.py - работа с БД (не менять)
✅ cache_service.py - работа с Redis (не менять)
✅ formatter.py - форматирование вакансий (можно стилизовать)
✅ search_keyboards.py - кнопки (можно менять текст)
✅ handlers/*.py - обработчики (мини-правки OK)


ПЕРЕД ЗАПУСКОМ ПРОВЕРЬ:
=======================

☑️ .env файл в корне проекта (НЕ env.env, ровно .env)
